📊 À propos du projet

Ce projet a été développé dans le cadre d'un Projet de Fin d'Études (PFE) de licence. Il démontre la faisabilité d'une solution IoT complète pour la surveillance industrielle à distance.


Technologies utilisées :

  • 🐍 FastAPI (backend REST)
  • 💾 InfluxDB Cloud (base de données temps réel)
  • 🎨 HTML5/CSS3/JavaScript (frontend)
  • 📊 Chart.js (graphiques interactifs)
  • ☁️ Render + Netlify (déploiement cloud)

🎯 Objectif du projet

Fournir une solution accessible, modulaire et open source pour la collecte, le stockage et la visualisation de données industrielles en temps réel.

🌐 Accès à distance

Cette plateforme est accessible depuis n'importe où dans le monde. Les données sont mises à jour automatiquement toutes les 3 secondes.
